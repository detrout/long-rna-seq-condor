Development Install
===================

If you'd like to be able to save your changes to the .condor files

.. code-block:: console

    git clone https://github.com/detrout/long-rna-seq-condor
    cd long-rna-seq-condor
    git checkout -b v1.1 1.1
    python3 setup.py install --user
